# MarkView

A local-first markdown file viewer with theming and directory navigation.

Open a folder of `.md` files and read them with beautiful typography, syntax highlighting, and 17 themes. Everything happens in your browser — no files are uploaded anywhere.

## Usage

1. Download `markview.html`
2. Open it in your browser
3. Click **Open Folder** or drag markdown files onto the page

## Features

- 17 themes (Dracula, Monokai, Nord, Tokyo Night, Catppuccin, GitHub, and more)
- Folder browsing with full directory tree
- Syntax highlighting for 190+ languages
- Math equations, Mermaid diagrams, wiki links, YAML frontmatter
- Full-text search across all loaded files
- Command palette (`Ctrl+K`)
- Pinned files, settings, export to HTML/PDF/ZIP
- Works fully offline once loaded

## Privacy

Files are read locally and never leave your browser. No tracking, no analytics, no backend.
